h1. PopWindow

Most easy way to use divs or other element as tooltip style windows withou modal, popups or lightbox

h3. Usage

Javascript
<pre>
	<code>
		<script src="http://code.jquery.com/jquery-latest.js"></script>
		<script src="jquery.popwindow.js"></script>
		<script src="jquery.bgiframe.js"></script>		

		<script language="javascript" type="text/javascript">

			$(document).ready(function(){
		  	$("#show").popwindow("#popup_div1");
		  });
   
		</script>
	</code>
</pre>

XHTML
<pre>
	<code>
		<input type="button" id="show" value="Show Message in Bottom"/>
		<div id="popup_div1" class="popup_msg">
		Popup Message<br /> 
		Message goes here <br /> 
		Message goes here
		</div>
	</code>
</pre>

CSS
<pre>
	<code>
		.popup_msg{ display:none; }
	</code>
</pre>

h3. Possible Params
<table border="1px" cellpadding="5">
	<thead>
		<tr>
			<th>Param</th>
			<th>Values</th>			
			<th>Desc</td>
		</tr>
	</thead>
	<tbody>
		<tr>
			<td>position</td>
			<td>"bottom","right","top","left"</td>			
			<td>display hidden element in passed position</td>
		</tr>
		<tr>
			<td>offsetTop</td>
			<td>Any number value (default is 10)</td>			
			<td>offset for top</td>
		</tr>		
		<tr>
			<td>offsetSides</td>
			<td>Any number value (default is 0)</td>			
			<td>offset for sides</td>
		</tr>
		<tr>
			<td>transitionSpeed</td>
			<td>"fast","slow" (default is "fast")</td>			
			<td>speed of fade effect</td>
		</tr>
				
	</tbody>
</table>
